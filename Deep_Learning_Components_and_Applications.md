# Deep Learning: Components, Applications, and Use Cases

## What is Deep Learning?

Deep Learning is a subset of Machine Learning that uses Artificial
Neural Networks (ANNs) with multiple hidden layers to automatically
learn patterns from data.

## AI Hierarchy

-   Artificial Intelligence (AI)
    -   Machine Learning (ML)
        -   Deep Learning (DL)

## Why Deep Learning?

Traditional ML often requires manual feature engineering. Deep Learning
automatically learns useful features from raw data.

## Main Components

1.  Input Layer
2.  Hidden Layers
3.  Neurons
4.  Weights
5.  Bias
6.  Activation Functions (ReLU, Sigmoid, Tanh, Softmax, GELU)
7.  Loss Function
8.  Optimizer (SGD, Adam, RMSProp)
9.  Backpropagation

## Training Workflow

Input Data → Forward Pass → Prediction → Loss Function → Backpropagation
→ Optimizer → Updated Weights

## Popular Architectures

-   ANN
-   CNN
-   RNN
-   LSTM
-   Transformers
-   Autoencoders
-   GANs

## Applications

-   Computer Vision
-   Natural Language Processing (NLP)
-   Speech Recognition
-   Recommendation Systems
-   Healthcare
-   Autonomous Vehicles

## Advantages

-   Automatic feature learning
-   High accuracy
-   Excellent for images, text, and audio

## Limitations

-   Large datasets required
-   High computational cost
-   Longer training time

## Key Takeaways

-   Deep Learning is a subset of Machine Learning.
-   It learns features automatically using neural networks.
-   Transformers are one type of Deep Learning architecture.
-   Deep Learning powers LLMs, computer vision, speech recognition, and
    many modern AI systems.
